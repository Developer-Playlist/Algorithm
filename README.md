# branch 생성 네이밍 규칙

```
git branch <성명>/<문제_제목>-P.<문제 페이지수>
```
</br>
예시
</br>

```
git branch 지기성/치킨_배달-P.332
```

- 성명: `성`, `이름` 모두 작성
- 문제 제목의 띄어쓰기는 `_`(underscore)로 연결
- 문제 제목과 페이지 수는 `-`(hyphen)으로 연결 
- 페이지의 약자는 대문자 `P`로 작성
- P 뒤에는 `.`(period) 붙이기
